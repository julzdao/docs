> **Azure Fundamentals az-900**\
> Certification for demonstrating foundational knowledge of cloud concepts in general and Microsoft Azure in particular. This certification is a common starting point in a journey towards a career in Azure.

The exam consists of three clearly differentiated sections:

1. **Describe Cloud Concepts (25-30%)** :: Describe basic concepts about how cloud computing works, the models used, and even learn how companies make money using these cloud pricing models. 
2. **Describe Azure Architecture and services (35-40%)** :: This section focuses on understanding the different services that Azure offers in relation to cloud computing. Explore the product itself.
3. **Describe Azure management and governance (30–35%)** :: Explore how Azure's economics and policies affect it in general. 

This certification gives us a foundation in Compute, Networking, and Storage. 

In relation to our Farming IoT project, we will use Azure to communicate, store, and distribute our data across various ecosystems. 

It will also help us become familiar with the opportunities we have to connect our data on Hydroponics with **Machine Learning** models. 

> **Focus on Security** 
> One of the reasons why Microsoft Azure is trusted by high-profile companies is its focus on security. More than 10,000 cybersecurity experts are responsible for protecting the data that flows through this software. Cool!

In conclusion, this certification is not only necessary for my profile, 
but also essential for our project.
