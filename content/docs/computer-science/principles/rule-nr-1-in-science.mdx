## Importance of Robust Documentation

1. **Improve Collaboration**: The docs help teams be on the same page, being aligned in a multi-crossed way.
2. **Scalability**: A well documented project is able to scalate and grow properly.
3. **Trust**: By adding sources to the documentation, information is much more reliable.

## Key tenets of documentation

1. Keep it **Clear**.
2. Keep it **Concise**.
3. Keep it **Structured**.
