## Introduction

FarmOrchestra is a **documentation vault**, created to share knowledge about Computer Science, Farming, Philosophy or even Jazz. 
A Digital Garden full of connections between concepts and ideas. 

This Computer Science vault consists of different parts:

<Cards>

<Card title='Foundations'>

Learn about Design Patterns, Clean code, Habits or Complex concepts. 

</Card>

<Card  title='Development'>

Programming Technical Knowledge base with notes and concepts about languages, frameworks or best practices. 

</Card>

<Card  title='Cloud'>

Concepts from Cloud Development focused to handle IoT applications. 

</Card>

</Cards>
