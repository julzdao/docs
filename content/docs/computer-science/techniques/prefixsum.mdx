The PrefixSum technique consists of creating a final array with the previous sum of elements in each position of the array.
It is similar to representing the snowball effect using an algorithm.

```java
prefixSum[i] = arr[0] + arr[1] + … + arr[i]
```

## Example code

```java
class PrefixSumExample
{
    public static int[] ComputePrefixSum(int[] arr)
    {
        int n = arr.Length;
        int[] prefixSum = new int[n];
        prefixSum[0] = arr[0];
        for (int i = 1; i < n; i++)
        {
            prefixSum[i] = prefixSum[i - 1] + arr[i];
        }
        return prefixSum;
    }
    static void Main()
    {
        int[] arr = { 2, 4, 6, 8 };
        int[] prefixSum = ComputePrefixSum(arr);
        Console.WriteLine("Prefix Sum Array:");
        foreach (var sum in prefixSum)
        {
            Console.Write(sum + " ");  // Output: 2 6 12 20
        }
    }
}
```
