There is a magical thing about learning programming.\
It Is like learning how computers think.

If we follow the principle of\
"Be Water my friend" from Bruce Lee,\
where we become one with the activity that we are doing,\
we can reach the so-called "Flow" state.

For this to happen while programming,\
one may need to start thinking like the program or language itself.
